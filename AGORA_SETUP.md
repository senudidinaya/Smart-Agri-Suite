# Agora Configuration and Setup Guide

## Prerequisites

- Agora.io account (sign up at https://www.agora.io)
- React Native project with Agora SDK installed
- Server-side token generation endpoint
- Node.js 12+ for backend token generation

## Step 1: Agora Console Setup

### 1.1 Create an Agora Account
1. Visit https://www.agora.io
2. Sign up for a free account
3. Create an organization
4. Create a project for Smart Agri Suite

### 1.2 Get Your App ID

1. Go to [Agora Console](https://console.agora.io/)
2. Navigate to **Project Management** → **Project List**
3. Find your Smart Agri Suite project
4. Copy the **App ID** (it's a 32-character string)
5. Store it securely in your `.env` file:

```env
# .env (Main Frontend)
AGORA_APP_ID=YOUR_32_CHAR_APP_ID

# Required for mobile builds
# .env.local or eas.json
EXPO_PUBLIC_AGORA_APP_ID=YOUR_32_CHAR_APP_ID
```

### 1.3 Get Your App Certificate

For production, you need an App Certificate for secure token generation.

1. In Agora Console, click the **Gear icon** next to your project
2. Click **Edit** to reveal the **Primary Certificate**
3. Copy and store it securely (NEVER commit to git):

```env
# Backend only - NEVER expose to client
AGORA_CERT=YOUR_64_CHAR_CERTIFICATE
AGORA_APP_ID=YOUR_32_CHAR_APP_ID
```

⚠️ **IMPORTANT**: Never expose App Certificate to the client. Always generate tokens on the server.

## Step 3: Server-Side Token Generation

The frontend needs RTC tokens generated by your backend. Create an endpoint:

### Node.js Example

```javascript
// backend/routes/agora.js
const express = require('express');
const { RtcTokenBuilder, RtcRole } = require('agora-access-token-builder');

router.post('/generate-token', async (req, res) => {
  const { channelName, uid, role = 'publisher' } = req.body;

  if (!channelName || !uid) {
    return res.status(400).json({ error: 'Missing parameters' });
  }

  try {
    const appId = process.env.AGORA_APP_ID;
    const appCertificate = process.env.AGORA_CERT;
    
    // Token expires in 1 hour
    const expirationTimeInSeconds = 3600;
    const currentTimestamp = Math.floor(Date.now() / 1000);
    const privilegeExpiredTs = currentTimestamp + expirationTimeInSeconds;

    // Map frontend role to Agora role
    const agoraRole = role === 'host' 
      ? RtcRole.PUBLISHER 
      : RtcRole.SUBSCRIBER;

    // Generate token
    const token = RtcTokenBuilder.buildTokenWithUid(
      appId,
      appCertificate,
      channelName,
      uid,
      agoraRole,
      privilegeExpiredTs
    );

    res.json({ 
      token,
      appId,
      uid,
      channelName,
      expiresIn: expirationTimeInSeconds,
    });
  } catch (error) {
    console.error('Token generation failed:', error);
    res.status(500).json({ error: 'Failed to generate token' });
  }
});
```

### Python Example (Flask)

```python
# backend/agora_service.py
from agora_token_builder import RtcTokenBuilder, Role_Rtc
import os
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/api/agora/generate-token', methods=['POST'])
def generate_token():
    data = request.json
    channel_name = data.get('channelName')
    uid = data.get('uid')
    role = data.get('role', 'publisher')
    
    if not channel_name or uid is None:
        return jsonify({'error': 'Missing parameters'}), 400
    
    try:
        app_id = os.environ.get('AGORA_APP_ID')
        app_certificate = os.environ.get('AGORA_CERT')
        
        expiration_time_in_seconds = 3600  # 1 hour
        
        role_obj = Role_Rtc.PUBLISHER if role == 'host' else Role_Rtc.SUBSCRIBER
        
        token = RtcTokenBuilder.build_token_with_uid(
            app_id,
            app_certificate,
            channel_name,
            uid,
            role_obj,
            int(time.time()) + expiration_time_in_seconds
        )
        
        return jsonify({
            'token': token,
            'appId': app_id,
            'uid': uid,
            'channelName': channel_name,
            'expiresIn': expiration_time_in_seconds,
        })
    except Exception as e:
        print(f'Token generation failed: {e}')
        return jsonify({'error': 'Failed to generate token'}), 500
```

## Step 4: Frontend Configuration

### 4.1 Create API Service

```typescript
// frontend/src/services/agoraService.ts
import axios from 'axios';

const API_BASE = process.env.REACT_APP_API_URL || 'http://localhost:8000';

export interface TokenResponse {
  token: string;
  appId: string;
  uid: number;
  channelName: string;
  expiresIn: number;
}

export async function getAgoraToken(
  channelName: string,
  uid: number,
  role: 'host' | 'guest' = 'host'
): Promise<TokenResponse> {
  try {
    const response = await axios.post<TokenResponse>(
      `${API_BASE}/api/agora/generate-token`,
      {
        channelName,
        uid,
        role,
      },
      {
        headers: {
          'Authorization': `Bearer ${authToken}`, // your auth
        },
      }
    );
    return response.data;
  } catch (error) {
    console.error('Failed to get Agora token:', error);
    throw error;
  }
}

export async function logCallMetrics(callData: {
  channelName: string;
  uid: number;
  duration: number; // seconds
  bytesReceived?: number;
  bytesSent?: number;
  recordingPath?: string;
}) {
  try {
    await axios.post(`${API_BASE}/api/calls/metrics`, callData, {
      headers: {
        'Authorization': `Bearer ${authToken}`,
      },
    });
  } catch (error) {
    console.error('Failed to log metrics:', error);
  }
}
```

### 4.2 Create Voice Call Component

```typescript
// frontend/src/screens/VoiceCallScreen.tsx
import React, { useEffect, useState } from 'react';
import { View, Text, Button, StyleSheet, Alert } from 'react-native';
import { useAgora } from '@/src/hooks/useAgora';
import { getAgoraToken } from '@/src/services/agoraService';

export function VoiceCallScreen() {
  const [loading, setLoading] = useState(true);
  const [agoraConfig, setAgoraConfig] = useState(null);
  const { state, joinChannel, leaveChannel, toggleMute, startLocalRecording, stopLocalRecording, isRecording } = useAgora(agoraConfig);

  useEffect(() => {
    // Load Agora token when component mounts
    initializeCall();
  }, []);

  const initializeCall = async () => {
    try {
      setLoading(true);
      const userId = 12345; // Get from auth context
      const channelName = 'agricultural-support';
      
      const tokenData = await getAgoraToken(channelName, userId);
      
      setAgoraConfig({
        appId: tokenData.appId,
        channelName: tokenData.channelName,
        token: tokenData.token,
        uid: tokenData.uid,
      });
    } catch (error) {
      Alert.alert('Setup Failed', error.message);
    } finally {
      setLoading(false);
    }
  };

  const handleStartCall = async () => {
    const success = await joinChannel();
    if (!success) {
      Alert.alert('Call Failed', state.error || 'Could not join call');
    }
  };

  const handleEndCall = async () => {
    await leaveChannel();
  };

  const handleRecording = async () => {
    if (!isRecording) {
      const success = await startLocalRecording();
      if (!success) {
        Alert.alert('Recording Failed', 'Could not start recording');
      }
    } else {
      const filePath = await stopLocalRecording();
      if (filePath) {
        Alert.alert('Recording Saved', `File: ${filePath}`);
        // TODO: Upload to server
      }
    }
  };

  if (loading) {
    return <Text>Loading...</Text>;
  }

  return (
    <View style={styles.container}>
      <Text style={styles.title}>Voice Call</Text>
      <Text style={styles.status}>Status: {state.connectionState}</Text>
      
      <View style={styles.userInfo}>
        <Text>Remote Users: {state.remoteUsers.length}</Text>
        <Text>Muted: {state.isMuted ? 'Yes' : 'No'}</Text>
      </View>

      {!state.isJoined ? (
        <Button title="Start Call" onPress={handleStartCall} />
      ) : (
        <>
          <Button title={state.isMuted ? 'Unmute' : 'Mute'} onPress={toggleMute} />
          <Button 
            title={isRecording ? 'Stop Recording' : 'Start Recording'}
            onPress={handleRecording} 
          />
          <Button title="End Call" onPress={handleEndCall} color="red" />
        </>
      )}

      {state.error && <Text style={styles.error}>{state.error}</Text>}
    </View>
  );
}

const styles = StyleSheet.create({
  container: { flex: 1, padding: 20 },
  title: { fontSize: 24, fontWeight: 'bold', marginBottom: 20 },
  status: { fontSize: 16, marginBottom: 10 },
  userInfo: { marginVertical: 20 },
  error: { color: 'red', marginTop: 10 },
});
```

## Security Best Practices

1. **Never commit credentials**
   - Use .env files (added to .gitignore)
   - Use environment variables in CI/CD
   - Rotate App Certificate regularly

2. **Token Security**
   - Generate on server only
   - Include user identification
   - Set reasonable expiration (1-2 hours)
   - Implement token refresh

3. **API Security**
   - Require authentication for token endpoint
   - Rate limit token generation
   - Log token generation attempts
   - Monitor for abuse

4. **Recording Privacy**
   - Always ask user permission
   - Implement retention policies
   - Encrypt recordings at rest
   - Implement secure deletion
